Installation
------------

Using `PIP <https://packaging.python.org/en/latest/key_projects/#pip>`_:

.. code-block:: Bash

    pip install stpr


You can also check out the code directly from `GitHub <https://github.org/hategan/stpr>`_:

.. code-block:: Bash

    git clone git@github.com:hategan/stpr.git

    cd stpr

    pip install -e .
