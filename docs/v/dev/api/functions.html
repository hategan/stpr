Stpr Functions¶

run(fn) → object | None[source]¶

Runs a coroutine.

The coroutine (or coroutine function) is scheduled for execution and waited for synchronously. For example:

async def f(x: float) -> float:
    return x * x + 5

r = stpr.run(f(2))
print(r)  # 9

If fn raises an otherwise uncaught exception, this function re-raises that exception.

Parameters:

fn – A coroutine or coroutine function to run.

Returns:

This function returns the result returned by the coroutine.

Return type:

object | None

start(fn) → None[source]¶

Starts a coroutine.

This method schedules fn for execution. The argument must be either a coroutine or a coroutine function. This function returns immediately after scheduling fn. If fn raises an exception that is not caught, the exception is printed on the console.

Parameters:

fn – A coroutine or coroutine function to start.

Return type:

None

async_partial(f, /, *args, **kwargs)[source]¶

Like functools.partial(), except for async functions.

Parameters:

  • f – The function to apply partially.

  • args – The arguments to apply.

  • kwargs – The keyword arguments to apply.

Returns:

The function f with the specified arguments partially applied.
